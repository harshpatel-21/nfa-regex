# Readme

Node js version 20.19+ or 22.12+ is required since Vite build tool is being used. If unable to upgrade node version go to package.json and change "vite": "^7.3.1", to "vite": "^5.4.11" (this is compatible with nodejs18 which is what is on the university lab computers)

[install all packages (run this before running other commands)] npm install

[start running the localhost] npm dev

[to run tests] npm run test

[to view coverage] npm test -- --coverage
